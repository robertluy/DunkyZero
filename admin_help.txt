Если у вас есть ощущение, что бот работает некорректно технически, введите /start;
В случае наличия желания прочесть эту памятку снова, введите /help;
Для просмотра товаров в наличии нажмите кнопку 'Каталог', затем выберите в появившемся меню категорию, в случае наличия товаров в ней, вам высветятся карточки товаров;
Для добавления товара данной категории, вам будет предложено ввести его номер;
Для просмотра вашей корзины, нажмите кнопку 'Корзина', вам будут представлены выбранные вами товары, которые все еще есть в наличии;
Для получения возможностей администратора, нажмите 'Админ-панель';
При необходимости произвести рассылку всем пользователям бота, надмите кнопку 'Сделать рассылку' и введите сообщение;
Когда пользователь совершит свой выбор, уведомление об это придет в специальный чат. Скипируйте тег этого пользователя, вернитесь в чат с ботом и нажмите кнопку 'Взаимодействие с пользователем'. После этого вам откроется новое меню. Здесь вы сможете просмотреть корзину этого пользователя, вставив его тег, удалить товары из его корзины. Нажмите кнопку 'Удалить из корзины', вам будут предствлены 3 способа удаления: удалить все товары из корхины этого пользователя; удалить все товары из его корзины, имеющие данный номер; удалить из корзины первый добавленный товар этого номера;
Вы можете выйти из любого меню, нажав кнопку 'Отмена'.
